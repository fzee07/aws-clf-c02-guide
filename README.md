## What's Inside

This is Book 1 of the **AWS Mastery Series — From Zero to Developer Associate**, covering all four domains of the AWS Certified Cloud Practitioner (CLF-C02) exam:

- **Domain 1** — Cloud Concepts (24%)
- **Domain 2** — Security and Compliance (30%)
- **Domain 3** — Cloud Technology and Services (34%)
- **Domain 4** — Billing, Pricing and Support (12%)

Includes exam traps, cheat sheets, backend developer angles, and plain-English explanations for every AWS service on the syllabus.

## Why This Guide

- **Single file** — one HTML page, no dependencies, works offline
- **Exam-focused** — covers exactly what AWS tests, nothing extra
- **Backend dev context** — Node.js-specific notes wherever relevant
